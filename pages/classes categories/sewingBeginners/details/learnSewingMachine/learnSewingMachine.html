Learn To Use A Sewing Machine:


This course teaches all you need to know about learning how to use your sewing machine, including threading, winding a bobbin, stitch length, stitch width, zig-zag, changing a needle and any troubleshooting.

This is an invaluable workshop for anyone thinking of buying a sewing machine or has bought one, but doesn't know where to start.
